# Car Race

Oyun bir tür şans oyunudur. Ekranda bulunan üç araçtan herhangi birini seçmeniz istenmektedir. Aracınızı seçtikten sonra **GAZLA** butonuna bastığınızda araçların hareket ettiğini göreceksiniz.
